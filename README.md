# limpieza-excel

Script simple en Python para limpiar archivos Excel.

## Qué hace
- Elimina la primera fila del archivo
- Limpia caracteres especiales
- Normaliza tildes y símbolos
- Unifica espacios
- Procesa columnas A y B
- Genera un Excel limpio sin encabezados

## Uso
```bash
python limpiar_excel.py --in archivo.xlsx --out limpio.xlsx



## Requisitos
- Python 3.9+
- pandas
- openpyxl
